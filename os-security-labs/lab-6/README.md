# Lab 6

### Introduction

This lab focuses on managing users and groups in Active Directory and understanding how access permissions are applied.

Students will create multiple users with different roles, including a normal user and an administrative user. The lab also demonstrates how to organise users into groups and assign permissions efficiently.

In addition, file and folder sharing will be configured, with emphasis on the difference between Share permissions and NTFS permissions and how they affect user access.
